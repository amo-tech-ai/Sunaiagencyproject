Create a modern consulting-style website design system inspired by the visual design language of BCG (Boston Consulting Group).

Goal:
Produce a clean, editorial, consulting-grade style guide suitable for a strategy consulting or AI advisory firm website.

Design principles:
minimal
professional
editorial layout
large whitespace
high readability
research-focused presentation.

Include the following sections in the style guide:

1. Typography
   Primary heading style with large editorial headlines
   Secondary headings for section titles
   Readable body text for long articles
   Caption and metadata text styles.

2. Color Palette
   Light neutral background colors
   Dark charcoal for primary text
   Subtle gray tones for UI elements
   Accent green used sparingly for highlights, links, and CTAs.

3. Layout System
   Centered content container
   Wide margins for readability
   12-column responsive grid
   Generous vertical spacing between sections.

4. UI Components
   Insight cards for articles and reports
   Service cards with minimal description
   Author profile cards
   Call-to-action banners
   Newsletter signup block
   Navigation header
   Minimal footer layout.

5. Image Style
   Large editorial photographs
   Technology illustrations
   Research charts and data visualizations
   Wide rectangular hero images with soft rounded corners.

6. Data Visualization
   Clean consulting-style charts
   Bar charts
   Process diagrams
   Strategic frameworks
   Infographics with minimal color usage.

7. Spacing Rules
   Large vertical rhythm between sections
   Consistent padding inside cards
   Balanced margins around images and text.

Visual tone:
consulting-grade
research-driven
strategic
minimalistic
premium corporate aesthetic.

The final design system should feel like a professional consulting publication similar to BCG, McKinsey, or Bain rather than a marketing landing page.
