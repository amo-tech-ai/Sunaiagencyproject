# Design Prompt — AI Technology Stack Cards (BCG Style)

Create a premium consulting-style technology stack section with **six categories of cards**.
Design should feel minimal, intelligent, and enterprise-grade similar to **BCG or McKinsey technology capability pages**.

## Layout

Responsive grid layout.

Desktop:
3 columns

Tablet:
2 columns

Mobile:
1 column

Each category contains **technology cards with icon, title, and short description**.

Cards should be:

* light background
* rounded corners
* subtle shadow
* generous whitespace
* clean typography.

Background surrounding cards:
dark charcoal or deep emerald gradient.

---

# Visual Style

Typography:
clean modern sans-serif
clear hierarchy between title and description.

Icons:
minimal line icons
monochrome or subtle emerald accent.

Color palette:
background: dark charcoal
cards: warm white
text: charcoal
accent: emerald green.

Spacing:
large padding inside cards
wide spacing between columns.

The final result should feel like a **premium AI consulting capability section** similar to BCG, Superside, or modern enterprise AI companies.
